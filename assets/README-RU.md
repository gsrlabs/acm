# 🛡️ acm — AmneziaWG CLI Manager

[![GitHub Release](https://img.shields.io/github/v/release/gsrlabs/acm?style=flat-square&color=blue)](https://github.com/gsrlabs/acm/releases)
[![Language](https://img.shields.io/badge/language-Go-00ADD8?style=flat-square&logo=go)](https://go.dev/)
[![OS](https://img.shields.io/badge/OS-Linux-orange?style=flat-square&logo=linux)](https://ubuntu.com/)

**acm** — это легковесная и удобная CLI-утилита на Go для управления вашими AmneziaWG (WireGuard) конфигурациями в терминале. Забудьте про ручной ввод длинных путей к конфигам — управляйте подключениями через интерактивное меню.

[Как правильно использовать AmneziaWG 2.0 через консоль](manual-awg-quick-ru.md)

## ✨ Основные возможности

* 🚀 **Интерактивное меню:** Выбирайте нужный конфиг нажатием одной цифры.
* 📂 **Умное хранение:** Все пути к конфигам сохраняются в JSON-базу данных (`~/.config/acm/`).
* 🛠️ **Быстрое управление:** Добавление новых конфигов в одно нажатие (`+`) и полное отключение всех интерфейсов (`-`).
* 📊 **Мониторинг:** Автоматический вывод статуса `awg show` прямо в интерфейсе.
* 🎨 **Красивый интерфейс:** Цветной вывод в терминале для удобства чтения.

---

## 🛠️ Системные требования

* **ОС:** Ubuntu / Debian / Pop!_OS (или любой дистрибутив с поддержкой `apt`).
* **Зависимости:** Утилита сама установит `amneziawg-tools`, если их еще нет в системе.

## 🚀 Быстрая установка

Самый простой способ установить `acm` — выполнить команду в терминале. Скрипт сам добавит PPA, обновит пакеты и скачает актуальный бинарник:

```bash
curl -sSL https://raw.githubusercontent.com/gsrlabs/acm/main/install.sh | sudo bash
```

После завершения установки просто введите:

```bash
acm
```

## 📖 Как пользоваться

### Первый запуск

Если список конфигураций пуст, программа предложит добавить первый сервер:

1.  Введите **абсолютный путь** к файлу `.conf` (например, `/home/user/vpn/germany.conf`).
2.  Введите **понятное имя** (например, `Германия`).

### Главное меню

  * **Цифры (1, 2, ...)** — Подключиться к выбранному серверу.
  * **`+`** — Добавить еще один конфиг в список.
  * **`-`** — Выполнить `down` для всех конфигов из вашего списка (полное отключение VPN).
  * **`0`** — Выход из программы.

## ⚙️ Технические детали

  * **Конфигурация:** Ваши настройки хранятся в файле `~/.config/acm/awg_configs.json`. Вы можете редактировать его вручную, если нужно быстро изменить пути.
  * **Автоматизация:** Утилита использует `sudo` для выполнения команд `awg-quick` и `awg show`. При первом действии система может запросить ваш пароль.
